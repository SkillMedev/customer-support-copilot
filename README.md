# Customer Support Copilot

**For support teams: resolve in one touch, escalate cleanly, deflect with self-service.** — built in-house by [Skill&nbsp;Me](https://skillme.dev).

Reach for this when you run a support queue and want every ticket to end in a resolution, not a follow-up. Seven skills cover the ticket lifecycle: draft empathetic one-touch replies, build a macro library that still sounds human, de-escalate angry customers and communicate refund decisions safely, write escalations engineering will actually act on, mine CSAT verbatims for the root causes worth fixing, turn resolved tickets into KB articles that deflect the next wave, and diagnose churn to build segmented retention playbooks. The outcome: faster resolutions, smarter escalations, and a queue that shrinks as self-service grows.

## Install

- **From the catalog:** [skillme.dev/pack/customer-support-copilot](https://skillme.dev/pack/customer-support-copilot) — install the whole pack into Claude in one step.
- **With the skills CLI:** `npx skills add SkillMedev/customer-support-copilot`
- **Manually:** copy any `skills/<slug>/SKILL.md` into your Claude skills directory.

## Skills in this pack

- **[Support Ticket Reply](skills/support-ticket-reply/SKILL.md)** — Draft empathetic, accurate, action-first support replies that resolve tickets in one touch.
- **[Support Macro Library](skills/support-macro-library/SKILL.md)** — Design a reusable macro and canned-response library that stays human and on-brand.
- **[Escalation Summary](skills/escalation-summary/SKILL.md)** — Write a tight escalation summary to engineering or management with repro steps, customer impact, and a clear ask.
- **[CSAT Root Cause Analysis](skills/csat-root-cause/SKILL.md)** — Analyze CSAT and NPS verbatims to find root-cause themes and prioritize fixes.
- **[Knowledge Base Article Writer](skills/kb-article-writer/SKILL.md)** — Turn a resolved support ticket into a clear, searchable knowledge-base or help center article.
- **[Refund and De-escalation](skills/refund-deescalation/SKILL.md)** — Handle angry customers, refund requests, and de-escalation with policy-safe, calm language.
- **[Churn Reduction](skills/churn-reduction/SKILL.md)** — Diagnoses churn root causes and designs intervention playbooks by segment.
